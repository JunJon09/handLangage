# Requirement


* requests 2.28.2
* selenium 4.8.2

# Installation


```bash
pip3 install requests
pip3 install selenium
brew list ffmpeg
pip3 install schedule
pip3 install mojimoji
pip3 install SpeechRecognition
```

# Usage


```bash
git clone https://github.com/handLangage/~
cd handLangage
python3 main.py
```
